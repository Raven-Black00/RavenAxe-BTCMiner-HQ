сЪ▒ What is RavenMiner HQ?

RavenMiner HQ is a real-time Bitcoin mining dashboard built for the AvalonMiner / RavenMiner family of ASIC miners. It monitors your miner over the local network API and presents all critical data in a dark, Norse-themed Tkinter GUI тАФ with runic motifs, Vegvisir watermarks, Valknut animations, and gold-on-purple aesthetics drawn from the Elder Futhark and Slavic symbolism.

Python 3.x Tkinter PIL / pystray Discord Alerts

API endpoint: http://<MINER_IP>/api/system/info
Tested on: AvalonMiner A14 series running RavenMiner firmware
